## Hey, I'm Rachel

Product Manager based in Brooklyn,building consumer products.

My background is in economics (UW, '23) and two-sided marketplaces. At Gymki, I shipped a consumer fitness app and partner tools from zero to launch across 70+ partners. Now I build my own products with Claude Code, Next.js, and Supabase.

### What I'm working on

**[Fynds AI](https://fynds-app-eta.vercel.app)** — AI shopping tool that translates how people actually describe what they want ("something for a rooftop dinner in July") into real product recommendations. Built with the Claude API as a translation layer between intent and inventory.

**[Diary App](https://diary-app-azure.vercel.app)** — Contextual journaling tool exploring what it looks like when a journal actually knows you. Moon-phase prompts, time capsule letters, birth chart engine using real astronomical calculations.

**[rachelmccarthy.io](https://rachelmccarthy.io)** — Personal site with a custom analytics stack (Supabase + middleware-based visitor tracking with RLS), D3 geo visualizations, and sections on essays, film photography, and music.
